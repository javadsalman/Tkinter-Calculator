# Calculater App developed with Python Tkinter

The calc_algorithm.py can be using as module for any project
---
```python3
>>> from calc_algorithm import CalcAlg
>>> calc = CalcAlg()
>>> calc.evaluate('12 - 5 * ( ( 13 + 2 ) / 10 )')
4.5
```
